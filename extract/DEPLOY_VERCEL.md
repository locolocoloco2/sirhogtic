# Despliegue en Vercel

Esta app es estática. No usa build.

## En Vercel

Framework Preset:
Other

Build Command:
dejar vacío o usar: echo "No build step required"

Output Directory:
./

Install Command:
dejar vacío

## Archivos que deben estar en la raíz del repositorio

index.html
styles.css
vercel.json
package.json
src/
assets/
supabase/

Si la página muestra texto como '# Sistema .DS_Store...', el repositorio quedó mal subido o Vercel no está apuntando a la raíz donde está index.html.
